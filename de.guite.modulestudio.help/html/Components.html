Introduction

ModuleStudio separates web applications into technical layers:

Models
Data layer defining the database structures and behaviour.
Controllers
Process layer defining use cases and user interaction.
Views
Output layer defining cosmetic aspects.

DSL

Meta model
Basic part of ModuleStudio, defines concepts and constraints
Editors
The graphical editors representing the GUI for modeling
Use general terms for modeling MVC applications
Generator
Adds technical details depending on the target system
Creates source code from built models
